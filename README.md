# Grayscale Images

### Convert images: make grayscaled version in browser.

You can browse and select your images with `Browse file...` button. The selected image is loading into the page. Bottom of this image, you can imediatelly see the graycaled version of the image in a separated area. You can download it to clicking on the `"blue cloud"` button.

### [DEMO](https://office.pergersoft.hu/grayscale-images/)

## Development server
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.12.

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.
